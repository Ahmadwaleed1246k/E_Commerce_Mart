E-Mart – My E-Commerce Android App 



I built E-Mart as a shopping app where buyers can browse products, search in real time, save favorites, add to cart, and chat with sellers — while sellers can easily add and manage their own products. The app uses Firebase for real-time data and authentication, Glide for loading images, and SQLite for local storage. I also added a smart fix that automatically converts Google Drive links into working image URLs so product photos actually show up. It's a complete marketplace solution that handles both sides smoothly
